# 多普勒风廓线激光雷达数据说明文档

## 一、数据背景说明

### 1.1 设备类型

本数据来自 **Molas3D 多普勒风廓线激光雷达**（Doppler Wind Profile Lidar），这是一种基于激光多普勒效应的主动遥感探测设备。

### 1.2 测量对象

主要测量大气中气溶胶粒子的运动速度，通过气溶胶作为示踪粒子反演大气风场信息。

### 1.3 工作原理

**多普勒测风原理**：

1. **发射激光**：激光雷达向大气中发射特定波长的激光束
2. **气溶胶散射**：激光束被大气中的气溶胶粒子散射
3. **多普勒频移**：运动的气溶胶粒子导致回波信号产生多普勒频移
4. **频移测量**：通过测量频移量计算气溶胶沿激光束方向的径向速度（RWS）
5. **风场反演**：通过多个方向的径向速度测量，可反演出三维风场

### 1.4 应用场景

- 气象预报与研究
- 风能资源评估
- 机场低空风切变监测
- 大气边界层研究
- 空气质量监测
- 城市风环境研究

## 二、字段含义解释

数据包含 **29 个字段**，以下逐一说明：

### 2.1 时空定位字段

#### 1. Timestamp（时间戳）
- **单位**：无（时间格式：YYYY/MM/DD HH:MM:SS.mmm）
- **物理意义**：激光雷达每次扫描测量的时间点
- **测量类型**：系统时间记录
- **示例值**：2025/10/05 00:00:00.934

#### 2. Azimuth(deg)（方位角）
- **单位**：度（°）
- **物理意义**：激光束在水平面内的指向角度，以正北为0°，顺时针方向增加
- **测量类型**：原始测量量（扫描控制参数）
- **数据范围**：本数据中为 52.008° ~ 62.005°（00941）和 227.045° ~ 251.004°（00943）
- **说明**：不同设备编号指向不同方位角扇区

#### 3. Elevation(deg)（仰角）
- **单位**：度（°）
- **物理意义**：激光束相对于水平面的俯仰角度
- **测量类型**：原始测量量（扫描控制参数）
- **数据范围**：
  - 00941: 1.683° 和 2.875°（两个固定仰角）
  - 00943: 6.784° 和 11.206°（两个固定仰角）
- **说明**：采用固定仰角扫描模式

#### 4. Distance(m)（距离）
- **单位**：米（m）
- **物理意义**：激光雷达沿发射方向到气溶胶散射点的距离
- **测量类型**：原始测量量（距离门设置）
- **距离范围**：100 m ~ 5166 m
- **距离门间隔**：17 m
- **距离门数量**：299 个

### 2.2 核心测量字段

#### 5. RWS(m/s)（径向风速）
- **单位**：米/秒（m/s）
- **物理意义**：沿激光束方向的风速分量（正值表示远离雷达，负值表示接近雷达）
- **测量类型**：原始测量量（基于多普勒频移计算）
- **数据范围**：-16.659 ~ -11.558 m/s（本数据中均为负值，表示风向雷达靠近）
- **说明**：这是激光雷达最核心的直接测量量

#### 6. CNR(dB)（载噪比）
- **单位**：分贝（dB）
- **物理意义**：回波信号强度与噪声之比，反映信号质量
- **测量类型**：原始测量量
- **数据范围**：2.336 ~ 18.509 dB
- **说明**：CNR越高，数据质量越好；通常需要CNR > -20 dB才认为数据有效

### 2.3 气象参数字段

#### 7. Temperature(°C)（温度）
- **单位**：摄氏度（°C）
- **物理意义**：雷达站点处的环境温度
- **测量类型**：辅助气象传感器测量
- **数据特征**：28.4°C（固定值，说明气象站测量频率较低）

#### 8. Relative Humidity(%)（相对湿度）
- **单位**：百分比（%）
- **物理意义**：雷达站点处的相对湿度
- **测量类型**：辅助气象传感器测量
- **数据特征**：84.7% ~ 85.0%（相对稳定）

#### 9. Air Pressure(kPa)（气压）
- **单位**：千帕（kPa）
- **物理意义**：雷达站点处的大气压强
- **测量类型**：辅助气象传感器测量
- **数据特征**：100.52 kPa（约标准大气压，1标准大气压 = 101.325 kPa）

### 2.4 风场反演字段

#### 10. Wind Speed(m/s)（风速）
- **单位**：米/秒（m/s）
- **物理意义**：通过多个径向速度反演得到的水平风速
- **测量类型**：派生变量
- **数据特征**：本数据中全部为 0.0，说明未进行风场反演或反演失败

#### 11. Wind Direction(deg)（风向）
- **单位**：度（°）
- **物理意义**：风的来向，气象学定义（正北为0°/360°，顺时针增加）
- **测量类型**：派生变量
- **数据特征**：本数据中全部为 0.0

#### 12. X Wind Speed(m/s)（东西向风速）
- **单位**：米/秒（m/s）
- **物理意义**：东西方向的风速分量（正值表示向东，负值表示向西）
- **测量类型**：派生变量
- **数据特征**：本数据中全部为 0.0

#### 13. Y Wind Speed(m/s)（南北向风速）
- **单位**：米/秒（m/s）
- **物理意义**：南北方向的风速分量（正值表示向北，负值表示向南）
- **测量类型**：派生变量
- **数据特征**：本数据中全部为 0.0

#### 14. Z Wind Speed(m/s)（垂直风速）
- **单位**：米/秒（m/s）
- **物理意义**：垂直方向的风速分量（正值表示上升，负值表示下沉）
- **测量类型**：派生变量
- **数据特征**：本数据中全部为 0.0

### 2.5 系统状态字段

#### 15. Mode（模式）
- **单位**：无（数值代码）
- **物理意义**：雷达工作模式标识
- **数据特征**：本数据中全部为 0.0

#### 16. Main（主通道）
- **单位**：无
- **物理意义**：可能表示主通道状态或标识
- **数据特征**：本数据中全部为 -0.01

#### 17. Serial Number（序列号）
- **单位**：无
- **物理意义**：数据点序列编号或设备序列号
- **数据特征**：本数据中全部为空值

#### 18. BKS
- **单位**：无

#### 19. Index（索引）
- **单位**：无
- **物理意义**：数据索引或计数器

### 2.6 光谱参数字段

#### 20. Spectrum Width(m/s)（谱宽）
- **单位**：米/秒（m/s）
- **物理意义**：多普勒速度谱的宽度，反映风速的湍流程度
- **测量类型**：派生变量（从多普勒谱计算）
- **数据特征**：本数据中全部为 0.0

#### 21. Spectrum Intensity(dB)（谱强度）
- **单位**：分贝（dB）
- **物理意义**：多普勒速度谱的强度峰值
- **测量类型**：原始测量量
- **数据范围**：142.19 ~ 147.21 dB

#### 22. Craft Broaden(m)（仪器展宽）
- **单位**：米（m）

#### 23. Craft RWS(m/s)（仪器径向风速）
- **单位**：米/秒（m/s）
- **物理意义**：扣除仪器影响后的径向风速，或仪器校准相关参数
- **测量类型**：派生变量
- **数据特征**：与 RWS 值相同或相近

### 2.7 大气光学参数

#### 24. Beta（后向散射系数）
- **单位**：无量纲或 m⁻¹·sr⁻¹
- **物理意义**：大气气溶胶的后向散射系数，表征大气散射能力
- **测量类型**：派生变量
- **数据特征**：极小值（-1.025504e-09 ~ 5.859511e-07），接近零

## 三、时间结构分析

### 3.1 时间字段

数据包含 **Timestamp** 字段，格式为 `YYYY/MM/DD HH:MM:SS.mmm`，精度达到毫秒级。

### 3.2 时间分辨率

通过分析数据发现：

- **每个时间戳对应 299 个数据点**（对应 299 个距离门）
- **相邻时间戳间隔**：约 0.8 ~ 1.1 秒不等
- **平均时间间隔**：约 1 秒

示例时间序列：
```
2025/10/05 00:00:00.934
2025/10/05 00:00:01.826  （间隔 0.892 秒）
2025/10/05 00:00:02.722  （间隔 0.896 秒）
2025/10/05 00:00:03.789  （间隔 1.067 秒）
2025/10/05 00:00:04.667  （间隔 0.878 秒）
```

### 3.3 扫描周期性

- **扫描模式**：每个时间点对应一个固定的（方位角，仰角）组合
- **单次测量**：包含从 100 m 到 5166 m 的 299 个距离门
- **采样频率**：约 **1 Hz**（每秒完成一个方向的全距离扫描）

### 3.4 数据采集时间

- 数据采集日期：2025年10月5日
- 采集开始时间：00:00:00（午夜）
- 在前 5000 行数据中包含 17 个时间戳，覆盖约 16 秒

## 四、空间结构分析

### 4.1 距离门设置

- **距离门间隔（Range Gate Spacing）**：17 m
- **起始距离**：100 m（第一个距离门）
- **最大探测距离**：5166 m（约 5.2 km）
- **距离门数量**：299 个

距离计算公式：
```
Distance = 100 + (n-1) × 17  （n = 1, 2, 3, ..., 299）
```

### 4.2 高度关系

通过仰角和距离可以计算几何高度：

**高度计算公式**：
```
Height = Distance × sin(Elevation)
```

以仰角 2.875° 为例：
- 100 m 距离 → 高度约 5.0 m
- 1000 m 距离 → 高度约 50.2 m
- 5000 m 距离 → 高度约 251 m

以仰角 11.206° 为例：
- 100 m 距离 → 高度约 19.4 m
- 1000 m 距离 → 高度约 194.3 m
- 5000 m 距离 → 高度约 971 m

### 4.3 水平距离

**水平距离计算公式**：
```
Horizontal Distance = Distance × cos(Elevation)
```

仰角较小（< 15°）时，水平距离与斜距接近。

### 4.4 空间覆盖

本数据不是完整的体扫描，而是：

- **扇形扫描**：在有限的方位角范围内扫描
- **多仰角扫描**：每个设备使用 2 个固定仰角
- **设备 00941**：方位角约 52° ~ 62°，仰角 1.683° 和 2.875°
- **设备 00943**：方位角约 227° ~ 251°，仰角 6.784° 和 11.206°

## 五、扫描模式判断

### 5.1 扫描模式分析

基于方位角（Azimuth）和仰角（Elevation）的分布特征，本数据的扫描模式为：

**修正的 DBS（Doppler Beam Swinging）模式 / 扇形 VAD 模式**

### 5.2 判断依据

#### 特征 1：固定仰角 + 多方位角
- **不是标准 VAD**：标准 VAD（Velocity Azimuth Display）需要在固定仰角下进行 360° 方位角扫描
- **不是 PPI**：PPI（Plan Position Indicator）需要水平面或近水平扫描（仰角接近 0°）
- **不是 RHI**：RHI（Range Height Indicator）需要固定方位角、变化仰角

#### 特征 2：数据特点

**00941 设备**：
- 2 个固定仰角：1.683°、2.875°
- 11 个方位角（约 52° ~ 62°，跨度约 10°）
- 每个时间戳对应一个固定的（方位角，仰角）组合

**00943 设备**：
- 2 个固定仰角：6.784°、11.206°
- 17 个方位角（约 227° ~ 251°，跨度约 24°）
- 每个时间戳对应一个固定的（方位角，仰角）组合

#### 特征 3：时间序列特征
- 每约 1 秒完成一个（方位角，仰角）方向的完整距离扫描
- 按顺序遍历不同的方位角
- 17 个时间戳对应 17 个不同的扫描方向

### 5.3 扫描模式结论

**扇形区域 DBS 扫描**：

1. **不是全方位扫描**：仅覆盖特定方位角扇区（而非 360°）
2. **多仰角探测**：使用 2 个固定仰角提高垂直分辨率
3. **时序扫描**：在指定扇区内按时间顺序逐个方向扫描
4. **应用目的**：
   - 可能是多台设备组网覆盖不同方位角区域
   - 针对特定风向或感兴趣区域的重点监测
   - 提高特定方向的时间分辨率

### 5.4 与标准扫描模式对比

| 扫描模式 | 方位角范围 | 仰角范围 | 本数据是否符合 |
|---------|-----------|---------|---------------|
| VAD | 360° | 固定（通常 15° ~ 75°） | 否（方位角不全） |
| PPI | 360° 或扇形 | 固定（接近 0°） | 否（仰角较大） |
| RHI | 固定 | 变化（0° ~ 90°） | 否（仰角固定） |
| DBS | 3-5 个固定方向 | 固定或多个固定值 | 是（扇形区域变体） |

## 六、数据质量与特殊情况

### 6.1 数据质量指标

- **CNR（载噪比）**：2.336 ~ 18.509 dB，大部分数据质量良好
- **低 CNR 区域**：远距离处 CNR 逐渐降低，数据质量下降

### 6.2 特殊值情况

1. **风场反演字段全为零**
   - Wind Speed = 0
   - Wind Direction = 0
   - X/Y/Z Wind Speed = 0
   - **原因**：数据为原始测量数据，未进行风场反演处理

2. **气象参数固定**
   - 温度、湿度、气压在整个扫描期间保持不变或微小变化
   - **原因**：气象站更新频率较低（通常为分钟级）

3. **空值字段**
   - Serial Number、Cloud Height、Aerosol Height 为空
   - **原因**：未探测到相应目标或功能未启用

### 6.3 数据有效性

- **有效测量数据**：RWS、CNR、Spectrum Intensity
- **需要后处理**：风速、风向等派生量需通过 VAD 或其他算法反演
- **辅助参考数据**：气象参数、大气边界层高度

## 七、数据使用建议

### 7.1 适用研究

1. **多普勒速度分析**：利用 RWS 研究径向速度特征
2. **信号质量评估**：利用 CNR 分析探测能力和大气状态
3. **扫描策略研究**：分析不同仰角和方位角的探测效果
4. **算法开发**：开发风场反演算法（VAD、VVP 等）

### 7.2 风场反演要求

要获得风速和风向信息，需要：

1. **至少 3 个不同方向的径向速度测量**
2. **合适的仰角**（通常 15° ~ 75° 用于 VAD）
3. **水平均匀性假设**：假设扫描范围内风场水平均匀

### 7.3 数据局限性

1. **方位角覆盖不完整**：仅覆盖局部扇区，难以进行完整的 VAD 反演
2. **仰角偏低**：1.683° ~ 11.206° 的仰角适合低空探测，但可能影响风场反演精度
3. **未经质控**：原始数据未进行质量控制和异常值剔除
